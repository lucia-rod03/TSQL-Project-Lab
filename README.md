# T-SQL Project Lab

En este repositorio donde documento soluciones técnicas y proyectos de gestión de datos utilizando **Microsoft SQL Server**.

El objetivo de este espacio es demostrar habilidades en la arquitectura de datos, limpieza (Data Cleaning) y procesos de transformación (ETL).

---

## Proyectos Disponibles

### 1. [ETL: Saneamiento y Normalización de Clientes](./01-Customer-Data-Cleaning)
**Problema:** : Una base de datos con registros duplicados, nombres mal formateados y fechas inconsistentes.
**Solución:** : Diseño de una capa de Staging para procesar datos "sucios" y transformarlos en una tabla maestra (DIM) con tipos de datos optimizados.

* **Técnicas:** : CTEs, Funciones de ventana, TRY_CAST, Normalización de Strings.
* **Estado:** : Finalizado
